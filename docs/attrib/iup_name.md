## NAME (non-inheritable)

Name of the control inside the dialog. Not related to [IupSetHandle](../func/iup_sethandle.md).

### Value

Text.

### Notes

The NAME value will be used by [IupGetDialogChild](../func/iup_getdialogchild.md) to find a child inside a dialog.

### Affects

All controls.
