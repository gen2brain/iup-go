## IupMapFont

Retrieves the name of a native font, given the name of the old IUP FONT names.
See the old [Font Names](../attrib/iup_font.md) table for a list of names. 

### Parameters/Return

    char* IupMapFont(const char *iupfont);

**Returns:** the name of the native font.
