## IupGetFunction

Returns the global callback associated to a name by [IupSetFunction](iup_setfunction.md).

### Parameters/Return

    Icallback IupGetFunction(const char *name);

**name**: name of the global callback.

**Returns**: the callback or NULL if not found.
