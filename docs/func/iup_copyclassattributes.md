## IupCopyClassAttributes

Copies all registered attributes from one element to another.
Both elements must be of the same class.

### Parameters/Return

    void IupCopyClassAttributes(Ihandle* src_ih, Ihandle* dst_ih);

**src_ih**: identifier of the source element.\
**dst_ih**: identifier of the destiny element.
